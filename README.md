# 🎬 NextUp

NextUp is a cross-platform mobile application built with **React Native** and **Expo** that helps users organize their movie and TV show experience in one place.

Instead of switching between streaming platforms or forgetting where you stopped watching, NextUp allows users to manage watchlists, track episode progress, discover new content, and even decide what to watch with friends.

---

## ✨ Features

- 🔍 Search movies and TV shows using TMDB API
- 🎞️ Browse trending and popular content
- 📌 Add titles to:
  - Want to Watch
  - Watching
  - Watched
- 📺 Track TV series episode progress
- 🎲 Random Picker for choosing what to watch
- 👥 Watch Together with friends
- 📱 QR Code friend sharing
- 💾 Local data persistence using AsyncStorage
- ⚡ Fast and responsive user interface

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/yourusername/NextUp.git
```

### Install dependencies

```bash
npm install
```

### Start the project

```bash
npx expo start
```

Run the application using:

- Android Emulator
- iOS Simulator
- Expo Go
